## Before publishing

- Run `npm install`.
- Run `npm run lint`.
- Run `npm test`.
- Run `npm run build`.
- Take screenshots and replace the screenshot placeholder in README.
- Deploy to GitHub Pages, Vercel, Netlify or another static host.
